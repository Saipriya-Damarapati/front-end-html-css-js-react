CSS Notes:
-----------
Three types of CSS
1. Inline CSS - it will be written inside the HTML tag
Example, <table border="1" cellpadding="10" cellspacing="5">
2. Internal CSS - inside head tag, using style tag, to create styles for our elements <style></style>
3. External CSS - separate css file with extension of .css, that will be imported into html
Example, style.css - imported in <head> tag

Selectors:
To apply separate styles
    Example, make every row different background colour
Five types
    1. Element - tag name
    2. Class - create class name (use .)
    3. Id - using id (use #)
    4. Universal - (use *)
    5. Grouping selectors - grouping few elements
